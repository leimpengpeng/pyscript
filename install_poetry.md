#### Installation

 $ curl -sSL https://raw.githubusercontent.com/python-poetry/poetry/master/get-poetry.py | python
 
#### poetry cmd accessible anywhere on the terminal

$ echo 'export PATH="$HOME/.poetry/bin:$PATH"' >> ~/.bashrc

#### Create app
$ poetry new app-name
#### Activate venv
$ peotry shell
#### install Dependency
$ peotry add [pkg-name]
